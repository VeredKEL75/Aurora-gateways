# Vacation Registration & Management System

Full-stack web application with role-based access (admin/user).
Admins can create, edit, and delete vacations, while users can browse and register.

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: JavaScript, HTML, CSS
- Authentication: Password hashing, role-based authorization

## Features
- Secure user registration & login
- Admin CRUD operations
- RESTful API
- Responsive UI

## Setup
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
